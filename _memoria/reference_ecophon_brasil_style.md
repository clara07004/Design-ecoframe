# Referência Visual — @ecophon_brasil

**Perfil:** https://www.instagram.com/ecophon_brasil/
**Uso:** Somente design visual e estilo de imagem — não de conteúdo ou produto.
**Detalhe completo:** `marca/DESIGN.md` → seção `image_style`

## Princípio central
Amarelo Ecophon (#F5C518) → Azul Ecoframe (#4A61A0) em todos os elementos de destaque.

## 3 tipos de fotografia

| Tipo | Mood | Uso |
|---|---|---|
| Dark lifestyle | Moody, pessoas em ação, atmosférico | Cover, problema |
| Architectural installation | Clean, natural, produto instalado | Solução |
| Product closeup | Técnico, detalhe de textura/perfil | Detalhe técnico |

## 8 elementos gráficos (já adaptados para Ecoframe)

1. **Box highlight** — retângulo Azul sólido atrás de keywords no texto
2. **Loop hand-drawn** — oval de traço fino Azul em volta de sujeitos/produto
3. **Bracket frame** — colchete geométrico branco no cover escuro
4. **Floating card** — card branco arredondado com soft shadow sobre foto
5. **Sound waves** — linhas curvas paralelas flanqueando elementos
6. **Icon circles** — círculos Azul com ícone branco (infográficos)
7. **Accent blob** — quarto de círculo Azul no canto → sinaliza CTA
8. **Pill CTA** — botão cápsula outline sem preenchimento

## 7 templates de layout

| Template | Background | Estrutura |
|---|---|---|
| T1 Cover | Full-bleed foto escura | Bracket frame + headline misto branco + pill CTA |
| T2 Problema | Off-white texturizado | Headline charcoal + body + boxes de highlight nas keywords |
| T3 Regra simples | Off-white texturizado | "Regra simples" + 2 linhas: ícone + label + detalhe |
| T4 Split dinâmico | Foto 60% esq + Azul 40% dir (borda curva) | Texto só no painel Azul |
| T5 Floating card | Full-bleed foto | Card flutuante + ondas sonoras laterais |
| T6 Solução produto | Off-white | Headline + arch-photo dir + material close-up base esq |
| T7 CTA | Fundo Azul sólido | Logo + headline curto + pill CTA + foto anterior vazando pela borda |

## Backgrounds

- **Off-white texturizado** (`#F5F5F0`) — slides educativos, com grid losango sutil opcional
- **Azul sólido** (`#4A61A0`) — CTA final, fechamento
- **Full-bleed foto escura** — covers, problema
- **Full-bleed foto clara** — solução com produto instalado
- **Split** — foto escura + painel Azul com borda curva

## Hierarquia tipográfica nas imagens

- Cover label: Montserrat 400 small
- Cover headline: Poppins 700 muito grande, branco
- Slide headline: Poppins 700 grande, charcoal escuro
- Body: Montserrat 400 médio
- Highlight: mesma fonte — só o background muda
- CTA: Montserrat 700 pequeno
