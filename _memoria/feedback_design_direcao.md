## Regra absoluta
**Todo slide precisa ter foto ou imagem de fundo.** Design só com texto é vetado — sem exceção, mesmo em slides educacionais ou de critério. Fundo vazio com texto é inaceitável.

**Why:** Usuária deixou claro: sem foto não existe post para a Ecoframe. Essa é uma exigência que não muda por briefing ou formato.

**How to apply:** Em qualquer layout (carrossel, post estático, capa), sempre partir de uma foto ou imagem de base. Texto sempre sobreposto a imagem — nunca flutuando em fundo sólido ou vazio.

---

## Estilo de referência aprovado

O carrossel `eficiencia-energetica` é a referência visual aprovada para todos os próximos carrosseis. Ver [[feedback-carousel-design-aprovado]] para os padrões exatos de código.

**Design premium, nichado, sofisticado** — para público de alto poder aquisitivo (arquitetos, construtores premium, proprietários de alto padrão), dentro dos padrões visuais já estabelecidos:
- Fotos reais de fundo, tratadas com overlay/gradiente
- Tipografia bold Poppins, grande e limpa
- Identidade Ecoframe consistente
- Logo 44px + separador + tagline "Esquadrias em PVC" no header

**O que não é essa direção:**
- Design vazio, editorial frio, espaço negativo excessivo — VETADO
- Layouts sem foto como base — VETADO
- Sketch arquitetônico / blueprint / grain / tipografia oversized viral — VETADO (testado, descartado pela usuária em 25/05/2026)
- Estilo Canva genérico ou template óbvio — VETADO
