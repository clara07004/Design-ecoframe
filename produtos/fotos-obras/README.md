## Como usar essas fotos

Conforme regra do CLAUDE.md, ao criar qualquer **conteúdo visual** para a Ecoframe (carrossel, post estático, story), priorizar **fotos reais de produto** no lugar de imagens geradas por IA.

Fluxo recomendado:
1. **Identificar o tipo de tomada necessária** (fachada / interior / detalhe / produto)
2. **Consultar o inventário abaixo** para encontrar o ID da foto desejada
3. **Baixar do Drive** via `download_file_content` ou diretamente pelo link de visualização
4. Para fotos HEIC (LeAnge - Mar): converter para JPEG antes do uso

## Subpasta LeAnge - Mar — projeto cliente

Subpasta `1_CrvL9sxza0gUG18hP0uegOS9p__2wDb` com **90 fotos HEIC** (~2-4MB cada). Inventário detalhado em `leange-mar/inventario.md`.

**Atenção:** formato HEIC (Apple) — não é diretamente visualizável em Windows. Para usar:
1. Baixar do Drive
2. Converter para JPEG via:
   - Online: cloudconvert.com / heicconvert.com
   - PowerShell: `magick input.heic output.jpg` (ImageMagick instalado)
   - macOS: Preview + Exportar

## Insights visuais aprendidos das fotos

A partir das 8 amostras visualizadas:

1. **Foco é residencial alto padrão** confirmado — mansões, casas com piscina, pé-direito duplo, fachadas trabalhadas
2. **Branco é padrão dominante** — confirma o relatório PSQ feito em "cor branca"
3. **Capacidade de cores** existe e é usada — cartela Lechler RAL + fotos com esquadrias pretas em pinázio
4. **Tipologias mais fotografadas:**
   - Porta de correr 4-6 folhas (integração casa-piscina/jardim)
   - Janelas de pinázio (estilo neoclássico e moderno)
   - Sacadas com guarda-corpo de vidro
5. **Construções tipicamente em alvenaria** (sistema convencional + contra-marco) — confirma o foco residencial vs. Steel Frame
6. **Padrão visual recomendado para conteúdo:**
   - Foto real + gradiente + texto (conforme regra de design aprovada)
   - Diversidade de obras — não usar sempre a mesma casa
   - Mostrar contexto arquitetônico (paisagismo, mobiliário) para dimensionar valor do projeto
