# Coleção Sincolors — Sincol

Linha de portas Sincol com **acabamento laqueado** em paleta colorida e neutra. Permite que a porta seja **elemento de cor** no projeto arquitetônico, ou que componha com paredes/marcenarias em projetos cromáticos.

## Quando especificar Sincolors

- Projetos onde a porta é **elemento de cor**, não apenas funcional
- Composições arquitetônicas com paleta cromática definida
- Ambientes contemporâneos minimalistas
- Hotelaria boutique com identidade visual única
- Residencial alto padrão com projeto de interiores integrado

## Tecnologia

- **Laqueado** — acabamento sólido com cor uniforme
- **U.V.** — pintura ultravioleta de alta resistência (em alguns tons)
- Sob consulta: cores especiais e tons exclusivos para projeto
