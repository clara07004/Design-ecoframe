# Sinkit — Kit Porta Pronta Sincol

Sistema de porta pré-montada da Sincol que entrega o conjunto completo (folha + batente + guarnição + dobradiças + fechadura) pronto para instalação em obra.

> Você acaba de adquirir o SINKIT — Kit Porta Pronta da Sincol. Produto único e de qualidade exclusiva.

## O que vem no kit

Conforme manual oficial Sinkit (versão FINAL 08/08/2023), o kit inclui:

- Folha de porta
- Batente
- Guarnição
- **Dobradiças**
- **Fechaduras**
- **Veda porta** (vedação inferior)

## Vídeo de instalação

O manual fornece QR Code para vídeo oficial de instalação. Ler a NBR 15530 antes de instalar — a Sincol responde pelo desempenho da porta quando instalada conforme manual.

## Modelos disponíveis (referência do manual)

A Sincol oferece variações Sinkit em:

- **Porta principal** (mais robusta, com fechadura específica)
- **Porta interna padrão**
- **Porta para áreas especiais** (umidade, fogo)
- **Diversas larguras:** 600, 700, 800, 900mm (folha)
- **Altura padrão:** 2.100mm

Em pedidos é especificado:
- Pedido / Lote / Sequência
- Localização (cidade, empreendimento)
- Pavimento, apto, cômodo
- Função (porta principal, sala, dormitório, banheiro)
