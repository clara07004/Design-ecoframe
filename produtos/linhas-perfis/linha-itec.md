# Linha iTEC — Sistema Leve

## Posicionamento

Sistema **LEVE** voltado para fabricação de esquadrias de **dimensões menores**. Utiliza marcos padrão de correr de 60mm e 109mm, com aplicação principal em sistemas de correr e maximar.

**Quando usar:** janelas residenciais menores, dormitórios, banheiros, áreas de serviço, hotelaria de quartos.

## Perfil principal

- **Folha:** FC-55 (37×55mm) — 695 g/m
- **Travessa:** TC-55 (37×55mm) — 635 g/m
- **Mão de amigo:** MAO-55 — 278 g/m
- **Marco 2 trilhos:** MC-60-2T (60mm × 38mm) — 915 g/m
- **Marco 3 trilhos:** MC-60-3T (60mm × 109mm) — 1.415 g/m
- **Batedor:** B-37 — 309 g/m

## Tipologias possíveis

- Janela / Porta de correr de **2, 3, 4, 6 folhas**
- Janela / Porta de correr com **tela mosquiteiro**
- Janela / Porta de correr com **veneziana** (1 vidro / 2 veneziana; 2 vidro / 1 tela)
- Janela / Porta de correr com **persiana de enrolar**
- Janela / Porta com **bandeira / peitoril**
- **Maximar** (com travessa ou com conexão)
- **Maximar de n folhas** (modulado)

## Acessórios e ferragens compatíveis

- Roldanas duplas com balancim (Detoni)
- Fecho concha 1 automático (lado esquerdo/direito) (Fermax)
- Maçaneta para correr / longa / com chave / dupla
- Cremona 200/400/1000/1400/1800/2100mm
- Trilho de alumínio (SBM)
- Tela mosquiteiro
- Calço de vidro
- Escovas Schlegel
- Tapa-deságue / tampa de canal de drenagem

## Quando NÃO usar iTEC

- Vãos acima de 1600mm de altura
- Necessidade de sistema oscilobatente ou de abrir tradicional
- Grandes vãos com alta exigência de carga de vento
- Edifícios acima de 30 pavimentos em regiões IV e V sem reforço externo

Para esses casos, escalar para **euroTEC** (intermediário) ou **TECplus-100** (pesado).

## Cor padrão

Branco (cor padrão de fábrica testada laboratorialmente).
